基于Hexo框架快速Blog搭建

什么是Hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

搭建步骤

下载安装VScode

Visual Studio Code - Code Editing. Redefined

下载安装Node.js

LTS版本即可

Node.js

下载安装Git for windows

Git - Downloads (git-scm.com)

注册GitHub个人账户

https://github.com

在磁盘中新建一个文件夹

  • 这个文件夹用于存放和管理Blog

  1. 比如我在D:\gitblog\下创建了一个BlogDemo文件夹

  2. 打开VScode，ctrl+o打开这个文件夹D:\gitblog\BlogDemo

  3. 在VScode中，ctrl+,进入设置搜索terminal.integrated.shell.windows

  4. 打开settings.json进行编辑

    将你Git安装目录下bin文件夹中的bash.exe的路径填入后覆盖ctrl+s保存退出

    {
    "terminal.integrated.shell.windows": "xxxxxx\\Git\\bin\\bash.exe"
}

  5. ctrl+shift+`打开一个bash终端准备就绪

更换node.js镜像源

由于在node的服务器在国外下载速度很慢，所以我们用淘宝的镜像源方便后续安装其他依赖

在bash终端里依次输入以下命令

npm config set registry https://registry.npm.taobao.org

配置ssh

  1. 在bash终端里依次输入以下命令

git config –global user.name “你的GitHub用户名”

git config –global user.email “你的GitHub注册邮箱”

  1. 生成密钥

ssh-keygen -t rsa -C “你的GitHub注册邮箱”

此时，在用户文件夹下就会有一个新的文件夹.ssh，里面有刚刚创建的ssh密钥文件id_rsa和id_rsa.pub

生成文件夹的路径: C:\Users\xxxx(你的windows用户名)\.ssh

  1. 然后将公匙添加到github上

用户头像→Settings→SSH and GPG keys→New SSH key→将id_rsa.pub中的内容复制到Key文本框中，然后点击Add SSH key(添加SSH)按钮。

创建博客仓库

仓库名应该为：xxxxxx(你的github用户名).github.io比如，比如我的仓库名是：iceixqwq.github.io

安装 hexo

在bash终端里依次输入以下命令

npm install -g hexo-cli

使用以下指令验证hexo是否安装成功

hexo -v

初始化 hexo

在bash终端里依次输入以下命令

hexo init

生成静态页面

在bash终端里依次输入以下命令

hexo g

开启本地服务器

hexo s

打开浏览器，地址栏中输入：**http://localhost:4000/**应该可以看见刚刚创建的博客了

将hexo博客部署到github上

在VScode中修改当前文件夹下的配置文件_config.yml(hexo的配置文件)，修改deploy项的内容（在文件末尾），把xxxxxx换成你github的用户名，如下所示：

# Deployment
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
  type: git
  repository: https://github.com/xxxxxx/xxxxxx.github.io
  branch: master
# 将xxxxx换成你的github的用户名，注意冒号后的空格不要删除

ps：如果出现报错ERROR Deployer not found: git则

npm install –save hexo-deployer-git

部署hexo

在bash终端里依次输入以下命令

hexo d

等待半分钟后打开在游览器中输入http://xxxxxx.github.io就能看到博客了

为Blog添加主题（以next主题为例）

  • 附上Next主题项目地址：

theme-next/hexo-theme-next: Elegant and powerful theme for Hexo. (github.com)

下载主题

在bash终端里依次输入以下命令

git clone https://github.com/theme-next/hexo-theme-next themes/next
  • 如果速度太慢了就在网页里点绿色的code按钮下载ZIP然后解压改名成next放到/themes目录下

切换主题

  1. 编辑配置文件_config.yml(hexo的配置文件)文件，切换成Next主题
## Themes: https://hexo.io/themes/
#theme: landscape
theme: next
  1. next现在有4种主题风格，编辑配置文件themes\next\_config.yml(next的配置文件)文件，切换
# Schemes
# 默认为Muse
scheme: Muse
#scheme: Mist
#scheme: Pisces
#scheme: Gemini

完成上述操作之后，用hexo clean命令清除下缓存，之后hexo g生成静态网页，可以通过hexo s来进行预览，hexo d部署到github上。

next主题的安装和配置可参考hexo-theme-next/docs/zh-CN at master · theme-next/hexo-theme-next (github.com)
